# Export as ico cur ani

## About this script

This script exports sprite as windows icon and cursor files, `.ico` and `.cur`, `.ani`.

## Installation

1. Open your script folder. (File -> Scripts -> Open Scripts Folder)
2. Put "Export as ico cur ani.lua"
